数据库

主键 一个数据表只包含一个主键，你可以使用主键来查询数据
外键 用于关联两个表
复何健 将多个列作为一个索引键 一般用于复合索引
索引 快速访问数据库特定信息 是对数据库一列或多列的排序 类似书籍目录
参照完整性 数据必须存在

刚安装 密码为空 通过
mysqladmin -u root password “new_password” 修改
登录
mysql -u root -p
增加用户
grant all on test(表名？).* ‘yourname’@’localhost’ identified by ‘youpassword’;
